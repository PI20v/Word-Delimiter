Назначение ПО
Данный продукт необходим для автоматизации обработки текста.

Данное ПО позволит ограничивать слова в исходном тексте длиной выше заданного порога.

Продукт поддерживает ограничение любых слов в любом тексте, содержащем и слова, и символы.

Размер текста не имеет значения.

Разработка
Данное ПО разрабатывается на платформе .NET 5.0 с использованием языка программирования C#.

Создано в среде разработки Visual Studio 2022 preview.
